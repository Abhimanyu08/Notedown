# Python Basics

This blog post will teach you the basics of python

### Variable Initialization

You can initialize a variable in python like so

```python
a = 2
```

You can then print it as shown

```python
print(a)
```

You can do all kinds of things with this variable

```python
b = a*2
print(b)
```
